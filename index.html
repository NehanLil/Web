HTML Components:

  • !DOCTYPE html: Declares the document type and version of HTML.
  • html: The root element of the HTML document.
  • head: Contains meta-information about the HTML document.
  • meta: Provides metadata about the HTML document (charset, viewport).
  • title: Specifies the title of the HTML page.
  • body: Contains the visible page content.
  • table: Defines an HTML table.
  • thead: Groups the header content in a table.
  • tbody: Groups the body content in a table.
  • tr: Defines a row in a table.
  • th: Defines a header cell in a table.
  • td: Defines a data cell in a table.

CSS Properties (Inline Styles):

  • border: Sets the border of the table.
  • bordercolor: Sets the color of the table border.
  • border-collapse: Collapses the table borders into a single border.
  • bgcolor: Sets the background color of a table cell (deprecated, but used in your code).
  • rowspan: Makes a cell span multiple rows.
